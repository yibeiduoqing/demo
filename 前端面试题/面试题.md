## 2.json是什么
JSON 适用于 Ajax格式，原因是它使 JavaScript 对象和字符串值之间得以快速转换 JSON是一种传递对象的语法

JSON提供了stringify和parse方法

stringify  js=>json标准的字符串

parse  json=>js对象
